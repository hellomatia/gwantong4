- ## 프로젝트 설명
  - Web MVC Architecture를 이해하고 활용할 수 있다.
  - Web Back-End 기술인 Servlet / JSP(Java Server Page)를 이해하고 활용하여 웹 서버를 구축할 수 있다.
  - MVC 아키텍처를 활용하여 확장성과 편리한 유지보수 등 장점을 살려 좋은 웹 서버를 구축할 수 있다.
